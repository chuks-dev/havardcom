Introduction to css

CSS(Cascading Style Sheet)

Used to style HTML file.

They are 3 major was of styling html files using css, namely:

  1. Inline Styling (Not recommended)

    CSS in written in html tags

  2. Embedded Styling (Not so recommended)

    CSS is written inside a style tag in the head of the html file

  3. External Styling (recommended)

    CSS is written in a separate .css file
