The Box Model

The box model says that all elements should be seen as boxes

The box can accept certain properties such as a:

  1. Margin

    The Margin is the outer space around an element

  2. Border

    THe Border is an stroke that is immediately after the element

  3. Padding

    The padding is a space inside the element, but before the border
